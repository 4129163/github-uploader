## 视频录制建议

### 工具推荐
- **录屏软件**: OBS Studio（免费）、Camtasia（付费）
- **剪辑软件**: 剪映、Premiere Pro
- **字幕工具**: 剪映自动字幕、Arctime

### 录制技巧
1. **分辨率**: 1920x1080 或更高
2. **帧率**: 30fps 以上
3. **声音**: 使用麦克风，减少环境噪音
4. **鼠标**: 使用高对比度鼠标指针
5. **标注**: 关键步骤使用箭头、高亮标注

### 发布平台
- Bilibili（程序员聚集地）
- 抖音/快手（简短版本）
- YouTube（国际观众）
