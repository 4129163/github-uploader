# 🚀 GitHub 自动上传工具

将 OpenClaw 源码自动上传到你的 GitHub 账号。

## 🚀 快速开始

### 1. 安装依赖

```bash
cd github-uploader
npm install
```

### 2. 获取 GitHub Token

1. 访问 https://github.com/settings/tokens
2. 点击 "Generate new token (classic)"
3. 勾选 **repo** 权限
4. 生成并复制 Token

### 3. 运行上传工具

```bash
npm start
```

## 📁 上传的文件

- 1-openclaw-main.zip (40MB) - 官方主仓库源码
- 2-openclaw-dashboard.zip (1.3MB) - Dashboard 源码
- 3-openclaw-dashboard-alt.zip (1.7MB) - Dashboard 另一版本
- 4-frontend.tar.gz (489KB) - 前端源码
- 5-backend-docs.tar.gz (11MB) - 后端文档
- 6-docker-deploy.tar.gz (7KB) - Docker 配置
- 7-full-package.tar.gz (36MB) - 完整资源包

## 🔧 环境要求

- Node.js >= 16
- npm

## 📝 说明

上传后可直接从你的 GitHub 账号下载所有资源。
