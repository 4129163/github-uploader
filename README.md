# GitHub Uploader

飞书妙搭 + CLI 工具 - GitHub 自动上传工具

## 📁 项目结构

```
github-uploader/
├── feishu-app/          # 飞书妙搭应用（NestJS + 飞书 SDK）
│   ├── src/             # TypeScript 源码
│   ├── cards/           # 消息卡片模板
│   └── package.json
│
└── cli-tool/            # 命令行工具
    ├── index.js         # 主程序
    └── package.json
```

## 🚀 快速开始

### 飞书应用

```bash
cd feishu-app
npm install
npm run dev
```

### CLI 工具

```bash
cd cli-tool
npm install
node index.js
```
